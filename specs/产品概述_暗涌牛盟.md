# 产品概述：【暗涌牛盟】（Hidden Stream Niu-Niu）

## 1. 项目愿景
打造一款极具沉浸感、房卡制、高频结算且具备高度“暗箱可控”能力的私域棋牌工具，满足特定私域圈层对强刺激、零成本起步（无底洞模式）以及可干预博弈的特殊需求。

## 2. 核心价值
- **绝对的掌控力（Master Control）**：为房主提供上帝视角的实时透视与干预手段（强制牛牛、全场点杀等），在不破坏游戏表面公平性的前提下实现精准控盘。
- **极致的感官刺激（UI/UX）**：通过“黑金赛博”视觉风格、动态筹码流向、以及拟真的“搓牌动画”与高频震动反馈，极大提升博弈过程中的肾上腺素。
- **高效的私域变现（Business Model）**：0分起步、1%精准抽水（仅针对赢家）、20局快速销毁结算，配合防赖账的战报与长图回放二维码，打造无缝的“开房-游戏-结算-收账”闭环。

## 3. 目标用户
### 主要用户群体
- **私域庄家/房主**：需要一款工具来组织私局，依赖抽水盈利，且需要一定的控场能力来规避风险或杀大户。
- **高频棋牌玩家**：追求刺激、喜欢看牌抢庄牛牛玩法的玩家群体。

### 使用场景
- 微信群私局约战，房主通过后台生成 6 位 Hash 码发至群内，玩家输入验证码进入房间。20局结束后自动出战报，房主依据战报和回放截图向玩家进行线下结算。

## 4. 系统模块
### 核心板块
1. **房卡与生命周期管理模块**：管理 6 位唯一 Hash 码的生成、激活、过期与销毁，控制 20 局的生命周期。
2. **核心玩法引擎（看牌抢庄 4+1）**：控制发牌（4张明+1张暗）、抢庄（1/2/4/8倍）、下注、比牌、0点起步记账与 1% 赢家抽水逻辑。
3. **前端渲染与动效模块（Vue全栈）**：实现 2人局/5人局的黑金赛博风格 UI、搓牌动画、震动反馈及金币粒子飞行动效。
4. **暗箱控牌引擎（Manipulation Core）**：包含预发牌池、请求拦截器（Interceptor）、实时换牌（Hot-swap）与平滑去重校验算法。
5. **上帝面板（Master Control）**：房主专属的透视与控制界面，包含强制牛牛、点杀等干预按钮。
6. **战报与防伪结算系统**：生成 20 局总榜单长图、二维码回放入口及微信通知。

## 5. 功能清单
### 第一阶段（MVP核心玩法与控牌）
- [ ] 房卡 Hash 码生成与校验系统。
- [ ] 基础的“看牌抢庄（4+1）”逻辑流转及 1% 抽水计算。
- [ ] 黑金赛博风格的基础 UI 框架（2人/5人布局适配）。
- [ ] 暗箱控牌核心：预发牌池、权重查询、发第5张牌时的拦截与平滑换牌算法。
- [ ] 房主视角的“上帝面板”基础版（透视与强制换牌）。

### 第二阶段（交互体验增强）
- [ ] “搓牌动画”实现：按住牌角滑动，露出牌面 1/4、1/2。
- [ ] H5 震动反馈集成（Vibration API，搓到大牌时高频震动）。
- [ ] 结算动效：金币粒子流向与 1% 抽水分流视觉效果。

### 第三阶段（结算与生态闭环）
- [ ] 20局总榜单生成（净分、抽水贡献、连胜记录）。
- [ ] 防赖账长图生成与二维码挂载。
- [ ] 扫码进入“1-20局牌局回放”功能。
- [ ] 微信推送对接（Server酱或企业微信API），向房主发送结算通知。

## 6. 验收标准
### 功能验收
- 控牌算法能在 50ms 内完成拦截、计算、去重与替换，玩家无任何网络延迟感知。
- 一局游戏结束后，所有玩家的分数总和加上抽水池分数，必须严格等于 0（零和博弈校验）。
- 房卡使用满 20 局后自动强制踢出并生成战报长图。

### 用户体验验收
- 黑金赛博风格 UI 适配移动端主流分辨率。
- 搓牌手感流畅，帧率保持在 60fps，H5 环境下震动 API 正常触发。
- 上帝面板的操作反馈即时生效，换牌结果与房主指令 100% 吻合且不出现重复牌。
